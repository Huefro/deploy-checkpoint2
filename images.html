This is where I learned about how to add images to my HTML document and alter it using CSS!


We add pictures to HTML documents by taking the source image address. A lot of times this can be done by right clicking on most images on google and selecting "Copy Image Address." That's what I do and voila, there's our picture! Say hello to the TBH creature!"


You can add videos too! This can be tricky to get the source file of, so what I do to make this work better is click on the share button of any YouTube video, and then click on "embed" where a big weird line of code shows up. It's one of the easiest ways that I know of to get the source code to any video.

Using the audio element, you can add audio files too. The problem is that it's more difficult to get the source code for various audio files, so just pretend i have one here.
